Instructions for Downloading

Right now this app is only available for Linux. You will have to clone the repository if you want to use it on a different OS.

  1. Download the `.AppImage` file.
  2. Open a terminal.
  3. Navigate to the directory where you downloaded the `.AppImage` file. For example, if you downloaded the file to your `Downloads` directory, you can use the command `cd ~/Downloads`.
  4. Add execute permissions to the `.AppImage` file with the command `chmod +x electron-ivy-lee-method-1.0.0..AppImage`.
  5. Now, you can run the `.AppImage` file by double-clicking it in your file manager. Depending on your desktop environment you may only be able to open the app with it in the file manager and not on the taskbar or desktop.

This is the first release of this Electron application. It's a to do list in the format of The Ivy Lee Method. When you download and run the app it will explain what The Ivy Lee Method is and how to use it. Its a great way to stay productive. Below are a list of the features the app provides.

  • Feature 1: A date box, to add the next days date.
  • Feature 2: A box to add tasks.
  • Feature 3: Delete button next to the written task and and error will occur if more than 6 tasks are written.
  • Feature 4: Add notes box.
  • Feature 5: A delete button next to the written note.
  • Feature 6: A pop up window that gives instructions on how to use the app.

Please download the .AppImage file to install the application on your Linux system.
